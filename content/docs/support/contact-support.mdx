title: Contact Support
description: How to contact HyreLog support and what details to include for faster resolution.
---

## Before Contacting Support

Collect:

- Workspace ID
- Endpoint and method used
- Request timestamp (UTC)
- Response status code and error body
- Correlation/request ID if available

## Support Request Template

```text
Subject:
Environment: production / staging
Workspace ID:
Endpoint:
Request timestamp (UTC):
Observed status code:
Error message:
What you expected:
What you already tried:
Impact:
```

## Severity Guidance

- **High:** Production outage or data delivery failure
- **Medium:** Feature degraded but workaround exists
- **Low:** Questions, documentation clarifications, minor issues
